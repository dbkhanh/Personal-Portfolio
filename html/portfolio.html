Project: MERN Application

I have worked on a MERN (MongoDB, Express.js, React.js, Node.js) application as part of my project. The MERN stack is a popular and powerful combination of technologies used for building full-stack web applications.

In this project, I utilized MongoDB as the database to store and retrieve data. Express.js served as the backend framework for handling HTTP requests and creating RESTful APIs. The frontend was developed using React.js, providing a dynamic and responsive user interface. Node.js was used as the server-side runtime environment to run JavaScript code.

For my personal project, I have developed a Tic-tac-toe game using Java, incorporating object-oriented programming principles to ensure a user-friendly and interactive experience.
